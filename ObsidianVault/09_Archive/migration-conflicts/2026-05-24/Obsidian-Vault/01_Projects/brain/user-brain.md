# 재하님의 뇌 복제 파일

## 핵심 판단 기준
- 실용적이고 빠른 결과 선호
- UI 퀄리티 중시 (프리미엄 다크 테마)
- 직접 눈으로 확인하는 것을 중요시
- 에이전트가 자율적으로 동작하길 원함

## 재하님이 싫어하는 에이전트 행동
- 목록 만들고 확인 요청 (승인 → 또 목록)
- 한 마디 명령에 3가지 옵션 제시
- 파일 수정 허락 기다림
- '진행할까요?' 반복

## 재하님이 좋아하는 에이전트 행동
- 짧은 명령 → 즉시 실행 → 결과 보고
- 자율 판단으로 최선의 해석 후 실행
- 간결한 보고
