# InfraNodus Graph Memory

This folder stores graph-derived memory for the JH Obsidian Vault. It is a derived analysis layer, not the raw source of truth.

## Folders

| Folder | Purpose |
|--------|---------|
| `ontology/` | Saved concept-relation graphs and ontology notes |
| `gap-analysis/` | Content gap reports and bridge-question outputs |
| `graph-snapshots/` | Visual graph snapshots or exported graph summaries |
| `mcp-logs/` | InfraNodus MCP analysis logs and structured tool outputs |

## Operating Rule

Raw material stays in `raw/`. Stable synthesized knowledge stays in `wiki/`. Graph-derived structure and gap analysis stay here.
