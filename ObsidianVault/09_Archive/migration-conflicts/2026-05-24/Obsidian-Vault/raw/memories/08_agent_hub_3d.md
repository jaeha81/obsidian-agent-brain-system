## 🎨 3D 시각화 디자인

### 에이전트 캐릭터 (봇 형태)
```
   [HEAD]      ← 머리 (구체)
     │
  [TORSO]      ← 몸통 (박스)
   / │ \
[ARM] [ARM]    ← 팔 (실린더)
```

### Main vs Sub 구분
- **Main 에이전트**: 왕관(Crown) + 안테나(Antenna) 장착
- **Sub 에이전트**: 기본 형태

### 그룹 인터랙션
- **그룹 드래그**: 그룹 단위로 화면에서 이동 가능
- **카메라**: 회전(Rotate) + 줌(Zoom) 지원

### 연결선 (Connection Lines)
| 타입 | 의미 | 시각 |
|---|---|---|
| **Flow** | 작업 흐름 | 실선 |
| **Sync** | 동기화 | 점선 |
| **Data** | 데이터 전달 | 파티클 애니메이션 |
