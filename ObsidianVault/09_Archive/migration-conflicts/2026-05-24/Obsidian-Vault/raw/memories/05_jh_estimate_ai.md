## 🤖 5-에이전트 파이프라인

```
입력 (평면도 / 사진 / 텍스트 요청)
    ↓
SCANNER
(도면/이미지 분석, 공간 정보 추출)
    ↓
ESTIMATOR
(공종별 물량 산출)
    ↓
PRICER
(단가 적용, 지역별/시장 가격 반영)
    ↓
VALIDATOR
(산출 검증, 이상값 감지)
    ↓
REPORTER
(최종 견적서 생성, 포맷 출력)
```

### 각 에이전트 역할

| 에이전트 | 입력 | 출력 |
|---|---|---|
| **SCANNER** | 평면도, 사진, 텍스트 | 공간 정보, 치수, 공종 목록 |
| **ESTIMATOR** | 공간 정보 | 공종별 물량 (m², m³, 개소) |
| **PRICER** | 물량 목록 | 공종별 단가 적용 금액 |
| **VALIDATOR** | 전체 견적 | 검증 결과, 이상값 플래그 |
| **REPORTER** | 검증된 견적 | 최종 견적서 (Excel/PDF) |

## 📈 Paperclip 벤치마크 기반 4대 업그레이드

경쟁 출품 전 Paperclip 벤치마킹 후 도출된 우선 개선 항목:

1. **Goal Tree View**: 에이전트 목표 구조 시각화
2. **Per-Agent Token Budget Tracking**: 에이전트별 토큰 사용량 추적 및 예산 관리
3. **Cron Trigger Layer**: 정기적 시장가 업데이트, 자동 재견적 트리거
4. **EstimateAI Template Packaging**: 공종별 견적 템플릿 패키징
