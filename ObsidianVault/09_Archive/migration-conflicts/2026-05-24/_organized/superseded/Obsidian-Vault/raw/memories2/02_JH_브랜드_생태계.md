> 인테리어 실무 경험을 기반으로 AI 자동화 시스템을 구축하고,
> 견적·상담·시공관리·콘텐츠·고객관리·수익화를 자동화하는 것.

## 7. JH Harness

| 항목 | 내용 |
|------|------|
| 성격 | 역할별 에이전트 조합 구조 |
| 철학 | 하나의 거대한 AI가 아닌, 역할별 에이전트 조합 |

**구조 개념**
- 하네스 = 공통 바디 역할
- 에이전트 = 교체 가능한 머리 역할
- 프로젝트별로 적합한 프레임워크 선택

**포함 시스템**
- 메인 관리 에이전트 (요구 분석 → 프레임워크 추천 → 개발 계획 → 보고)
- 전문 에이전트 (프론트/백엔드/DB/보안/UI/API/자동화/문서화/테스트/배포/수익화/인테리어 도메인)
- 검증 에이전트 (코드/보안/법적위험/UX/성능/구현가능성/MVP 기준)
- 기록 시스템 (개발로그/프롬프트/오류/결정사항/체크포인트 → GitHub/Obsidian/DB)

## 10. 브랜드 간 관계도

```
재하 (컨트롤 중심)
│
├─ 인테리어 사업
│   ├─ 스페이스819 (포트폴리오 / 상담)
│   ├─ 도연인테리어 (자동화 복사 적용)
│   └─ 더나은공간 (자문 / 임원)
│
└─ AI 자동화 사업
    ├─ JH Connect (플랫폼 브랜드 ⭐)
    ├─ JH.AI (오케스트레이터)
    └─ JH Harness (에이전트 구조)
```
