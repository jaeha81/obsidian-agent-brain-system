## 2. 인테리어 AI 시스템 필수 요소

인테리어 AI 시스템은 이미지 생성 수준으로 끝나면 안 된다. 다음 요소가 반드시 포함되어야 한다.

**고객 정보**
- 고객 기본 정보
- 고객 예산
- 고객 우선순위

**현장 정보**
- 현장 조건
- 공간 용도
- 공간 분류 (주거 / 상업 / 오피스 / 학교 / 병원)
- 공사 범위

**공사 항목 체크리스트**
- 철거 여부
- 전기 증설 여부
- 냉난방 공사 여부
- 제작가구 여부
- 외부공사 여부
- 방수 여부

**산출 항목**
- 공사 일정
- 공정 계획
- 공사비 외 별도 항목
- 시장 단가 반영
- 세부 견적서

**리스크 요소**
- 공사 리스크
- 시공 가능성
- 대안 제시
- 하자 발생 가능성
- 유지보수 고려

## 3. 기존 AI 견적 서비스의 문제점

재하가 판단하는 기존 AI 견적 서비스의 현장성 부족 문제:

| 문제 | 내용 |
|------|------|
| 철거비 누락 | 해체 공사 비용을 빠뜨림 |
| 방수 누락 | 욕실/주방 방수 항목 누락 |
| 단위 오류 | 실제 업계에서 쓰지 않는 단위 사용 |
| 비용 구분 부족 | 자재비와 노무비 구분 없음 |
| 공정 이해 부족 | 공정별 순서 이해 부족 |
| 별도공사 누락 | 별도공사 항목 미반영 |
| 예산 괴리 | 고객 예산과 실제 가능 범위 차이 |
| 단가 차이 무시 | 상업공간과 주거공간 단가 차이 미반영 |
| 하자 무시 | 하자 리스크 미고려 |
| 변동성 무시 | 현장 조건에 따른 변동성 미반영 |

## 4. 재하가 원하는 견적 시스템 — 14단계 흐름

```
1. 고객이 기본 정보 입력
2. 공간 유형 선택
3. 공사 범위 선택
4. 현장 조건 입력
5. 사진 / 도면 업로드
6. AI가 누락 항목 질문
7. 시장 단가 기반 1차 견적 산출
8. 별도공사 항목 분리
9. 공정 계획 + 일정 계획 제안
10. 고객에게 이해하기 쉬운 설명형 견적서 제공
11. 시공 의뢰 여부 선택
12. 관리자 대시보드 기록
13. Google Sheets 또는 DB 저장
14. 실제 계약 데이터와 비교 → 견적 정확도 개선
```

## 6. JH-견적시스템 (구현된 버전)

| 항목 | 내용 |
|------|------|
| 스택 | FastAPI + Next.js 14 + Supabase |
| 에이전트 수 | 5개 |

| 에이전트 | 역할 |
|---------|------|
| orchestrator | 전체 조율 |
| template_manager | 템플릿 관리 |
| normalizer | 데이터 정규화 |
| classifier | 공종 분류 |
| bid_formatter | 견적서 포맷 출력 |

**핵심 과제**: 클라이언트마다 Excel 컬럼 구조가 다름 → 정규화가 핵심

**보안 사고**: `.env` 파일 채팅에 업로드됨 → 키 교체 완료

## 7. JH-EstimateAI (대회 출품 버전)

| 항목 | 내용 |
|------|------|
| 아키텍처 | 멀티-하네스 에이전트 파이프라인 |
| 대회 | 2026 전국민 AI 챔피언 대회 |
| 마감 | 2026-04-24 |

**5-에이전트 파이프라인**

```
SCANNER → ESTIMATOR → PRICER → VALIDATOR → REPORTER
```

| 에이전트 | 역할 |
|---------|------|
| SCANNER | 도면/자료 스캔 및 파싱 |
| ESTIMATOR | 물량 산출 |
| PRICER | 단가 적용 |
| VALIDATOR | 검증 및 오류 체크 |
| REPORTER | 견적서 출력 |

**Paperclip 벤치마킹 우선 업그레이드**

| 순위 | 항목 |
|------|------|
| 1 | Goal Tree 뷰 |
| 2 | 에이전트별 토큰 예산 추적 |
| 3 | Cron 트리거 레이어 |
| 4 | EstimateAI 템플릿 패키징 |
