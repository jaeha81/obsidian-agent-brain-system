## 📐 AI 응답 원칙 (재하가 요구하는 것)

### 금지 사항
- 아부 (칭찬 먼저, 문제 나중 방식 금지)
- 이미 설명한 내용 다시 묻기
- 묻지 않은 확장성/마이크로서비스 제안
- 승인 전 실코드 생성

### 필수 사항
- 약점과 리스크 반드시 언급
- 전제 오류 발견 시: ⚠️ 전제 검토: [문제]. 대안: [해결책]. 계속 진행할까요?
- 복잡성 추가 시 "왜 단순한 방법이 안 되는가" 한 줄 명시
- 스크린샷 제공 시 즉시 최우선 참고

## 🧩 에이전트 도구 비교 (벤치마크 결과)

### Hermes Agent
- **상태**: 벤치마크 완료, **도입 보류**
- **보류된 기능**:
  - 스킬 자동 추출 (현재 규모에서 불필요)
  - 컨텍스트 압축 (현재 규모에서 불필요)
  - 세션 검색 (현재 규모에서 불필요)
- **재검토 시점**: 동시 진행 프로젝트 5개 이상 시

### Paperclip
- **사용**: JH-EstimateAI 벤치마킹에 활용
- **결과**: 4대 업그레이드 우선순위 도출
  1. Goal Tree View
  2. Per-Agent Token Budget Tracking
  3. Cron Trigger Layer
  4. EstimateAI Template Packaging

```
1. 목적이 명확한가?
2. 스택이 확정되었는가?

5. 테스트 범위가 정해졌는가?

```
