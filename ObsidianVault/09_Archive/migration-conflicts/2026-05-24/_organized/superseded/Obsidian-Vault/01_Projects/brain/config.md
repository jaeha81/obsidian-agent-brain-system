# JH Brain 설정

## 경로 매핑
- Obsidian 볼트: ~/JH-Brain/
- Windows 경로: C:\Users\user1\JH-Brain\
- 프로젝트 작업 폴더: ~/projects/ 또는 각 프로젝트 루트
- Brain System 대시보드: http://localhost:3457

## 규칙
- 모든 프로젝트의 research.md, plan.md는 projects/ 하위에도 복사본 유지
- 스킬 파일은 skills/ 폴더가 원본, 프로젝트의 .skills/는 심볼릭 링크
- 데일리 노트는 매일 자동 생성 (brain-daily.sh 연동)
- [[링크]]는 적극 활용 — 프로젝트 간 지식 연결이 핵심 가치

## Brain Bridge 연결
- Brain System 서버: http://localhost:3457
- JH Harness 연결: http://localhost:3456
- 브릿지 스크립트: ~/jh-brain-system/scripts/

## 토큰 최적화 설정
- 단순 작업 (파일 읽기, git 작업): Haiku 4.5 우선
- 일반 구현, 리서치: Sonnet 4.6
- 복잡한 아키텍처, 디버깅: Opus 4.6
- 캐시 TTL: 60초 (동일 검색 재호출 방지)
