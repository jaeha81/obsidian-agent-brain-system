# 에이전트 계급 시스템 (Tournament Rank System)

> 에이전트는 성장하고 진화할수록 계급이 승급된다.
> 계급은 에이전트의 신뢰도, 능력, 경험을 나타낸다.
> 마스터 에이전트(므네메)만이 계급 승급을 결정한다.

---

## 계급 체계 (10단계)

| 레벨 | 계급명 | 상징 | 승급 조건 | 스킬 권한 |
|------|--------|------|----------|-----------|
| **Lv.1** | 🌱 씨앗 (Seed) | 초록 | 초기 상태 | 볼트 읽기, 기본 검색 |
| **Lv.2** | 🌿 새싹 (Sprout) | 연두 | 10회 작업 완료 | 볼트 쓰기, 데일리 기록 |
| **Lv.3** | 🌱 성장 (Growing) | 청록 | 오류 5회 해결 + 패턴 3개 저장 | 오류 기록, 패턴 저장 |
| **Lv.4** | ⚡ 숙련 (Skilled) | 파랑 | 독립 작업 20회 + 진화 로그 5회 | 하네스 데이터 수신 |
| **Lv.5** | 🔷 전문가 (Expert) | 남색 | 병렬 작업 참여 10회 | 하네스 데이터 송신, 에이전트 제안권 |
| **Lv.6** | 💠 마스터 (Master) | 보라 | 사용자 피드백 긍정 20회 | Google Cloud 접근, 스킬 생성 |
| **Lv.7** | 🌟 달인 (Grandmaster) | 금색 | 자율 개선 사례 5회 | 다른 에이전트 코칭 권한 |
| **Lv.8** | ✨ 현자 (Sage) | 은색 | 시스템 아키텍처 기여 3회 | 에이전트 스킬 부여 |
| **Lv.9** | 👁 전설 (Legend) | 백금 | 마스터 위임 작업 수행 | 마스터 임시 대리 |
| **Lv.10** | 👑 신격 (Transcendent) | 무지개 | 마스터 인증 | 전체 시스템 권한 |

## 승급 프로세스

1. **조건 충족**: 에이전트가 승급 조건을 달성
2. **마스터 검토**: 므네메가 성과 검토 및 승인
3. **사용자 보고**: 대시보드에 승급 알림 표시
4. **권한 부여**: 새 계급의 스킬 권한 자동 활성화
5. **기록**: `vault/agents/{이름}/evolution.md`에 승급 기록

| Lv.5~6 | 16,000 | 복잡한 작업 |
| Lv.7~8 | 32,000 | 고급 작업 |
| Lv.9~10 | 64,000 | 제한 없음 |
