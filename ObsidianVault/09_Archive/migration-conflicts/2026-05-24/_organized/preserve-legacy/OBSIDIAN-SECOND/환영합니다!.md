# 나의 세컨드 브레인에 오신 것을 환영합니다

이곳은 **LLM Wiki** — JH의 복리 지식 베이스입니다.
LLM 에이전트가 소스를 읽고 위키를 직접 작성·유지합니다.

## 빠른 시작

```
/ingest [파일경로 또는 URL]   # 새 소스 추가
/query [질문]                  # 위키에 질문
/lint                          # 건강 검진
/status                        # 현재 통계
```

## 구조

- `wiki/` — LLM이 작성·유지하는 위키 (→ `wiki/index.md`부터)
- `raw/` — 불변 원본 소스 (아티클, 논문, 노트)
- `CLAUDE.md` — 운영 스키마 (LLM 행동 지침)
