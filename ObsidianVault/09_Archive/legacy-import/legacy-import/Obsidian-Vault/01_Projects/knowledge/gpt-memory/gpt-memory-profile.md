## AI 협업 철학 및 원칙

- **협업 파트너십 우선**: AI를 단순 도구가 아닌 파트너로 인식. 고부가가치 시스템 구축 → 수익은 자연히 따라오는 구조.
- **인간은 탐구, AI는 조력**: 기준점이 고정되지 않는 AI 시대에 인간의 역할은 탐구, AI는 방대한 데이터 서칭 및 조력.
- **창의적 다중 해석 선호**: 답변 정형화보다 다양한 경우의 수에서 나오는 다중 해석을 통해 창의적 사고 확장.
- **오케스트레이션 분리**: 재하는 전체 계획 수립 + 실행 오케스트레이션 담당, 구체적 개발(코딩/구현)은 에이전트에 위임.

### 4대 핵심 운영 시스템 (모든 개발·자동화·에이전트 운영 기본 규칙)
