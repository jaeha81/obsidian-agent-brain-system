# JH 수익 모델 프레임워크

> E1(무드보드 구독) + E4(인테리어 앱 수익화) + E7(번들+좌석구독) + E11(판매·구독 가격 뼈대) 통합 실행 문서

---

## 1. 핵심 수익 레버 3가지

```
레버 1: 단건 판매 (고마진, 즉시 수익)
레버 2: 구독 (반복 매출, 예측 가능)
레버 3: 번들 패키지 (ARPU 확대, 객단가 상승)
```

### JH 적용 매핑

| 서비스 | 레버 | 현재 상태 | 목표 |
|--------|------|---------|------|
| Express Mockup (인테리어 렌더) | 단건 + 구독 | 아이디어 단계 | 파일럿 착수 |
| 스나이퍼 구매대행 | 단건 수수료 | MVP v0.1 배포 | A/B 가격 테스트 |
| AI 자동화 컨설팅 | 프로젝트 단건 | Wishket 수주 | 구독형 리테이너 전환 |
| Bucky OS 솔루션 | 단건→SaaS | 개인 운영 | 화이트라벨/라이선스 |

## 2. 구독 가격 뼈대

### 기본 3-티어 구조

| 티어 | 가격/월 | 포함 내용 | 타겟 |
|------|--------|---------|------|
| **Starter** | ₩29,000 | 기본 기능 + 이메일 지원 | 개인/소호 |
| **Pro** | ₩59,000 | 전체 기능 + 우선 지원 + API | 중소사업자 |
| **Enterprise** | 협의 | 화이트라벨 + 전담 관리 | 기업 |

**연간 결제 할인**: 2개월 무료 (16.7% 절감) → 고객 락인 강화

### Express Mockup 단건 가격

| 상품 | 가격 | 마진율 |
|------|------|--------|
| 무료 미리보기 (워터마크) | ₩0 | - |
| 기본 2K 렌더 1장 | ₩15,000 | ~80% |
| 스탠다드 3장 + PSD | ₩35,000 | ~75% |
| 프리미엄 5장 + 수정 1회 | ₩65,000 | ~70% |

## 3. 번들 패키지 설계 (ARPU 확대)

### Simology 사례 적용 원칙

- 핵심 제품 + 보완 서비스 묶음
- 개별 합산 대비 20~30% 할인 (체감 가치↑)
- 파트너에게 마진 20~30% 배분 옵션

### JH 번들 후보

**Bundle A — 인테리어 AI 패키지** (소상공인 대상)
```
= Express Mockup Pro + AI 자재 추천 + 견적 자동화
가격: ₩89,000/월 (개별 합산 ₩120,000 대비 26% 할인)
```

**Bundle B — Wishket 자동화 패키지** (프리랜서 대상)
```
= 제안서 자동화 + 키워드 스코어링 + 알림 봇
가격: ₩49,000/월 (기존 ₩무료 → 유료 전환 실험)
```

## 4. 인테리어 앱 수익화 방향 (E4)

### 현재 상황
- 건축/인테리어 현장 경험 18년 보유
- AI 자동화 파이프라인 구축 완료
- 타겟: 소상공인 인테리어 사업자

### 수익화 3단계 로드맵

**1단계** (지금 ~ 1개월): 단건 Express Mockup 파일럿
- 카카오채널/Instagram DM으로 첫 5건 수동 처리
- 고객 피드백 → 체크리스트 정제

**2단계** (1~3개월): 자동화 퍼널 구축
- 랜딩페이지 → 이미지 업로드 → 무료 미리보기 → 유료 전환
- Stripe + 카카오페이 연동

**3단계** (3개월+): 구독 전환
- 월 5건 이상 주문 고객 → Pro 구독 제안
- 파트너 채널 (인테리어 소품 업체, 시공 업체)

---

## 5. 무드보드 구독 파트너 실험 (E1)

### 핵심 가설
> 무드보드 + AI 렌더를 결합한 구독 상품이 인테리어 사업자의 반복 구매를 유도할 수 있다.

### 파트너 채널 후보
- 인테리어 소품 업체 (렌더 결과에 자사 제품 노출)
- 가구/조명 제조사 (제품 비주얼 마케팅 활용)
- 인테리어 디자이너 커뮤니티 (화이트라벨 리셀링)

### 실험 설계
```
기간: 2주
대상: 인테리어 사업자 10~20명 (카카오채널/블로그 유입)
측정: 구독 전환율, 월 사용 빈도, 이탈율
성공 기준: 10명 중 3명 이상 2회차 결제
```
