### AutoRAG 2.0 조사 결과 요약

- **AutoRAG v0.3.22** (Marker-Inc-Korea, 오픈소스) — "2.0"은 공식 미출시, 팟캐스트에서 예고만
- **Gemini-RAG 대안으로 부적합**: AutoRAG는 RAG 파이프라인 최적화 도구, Gemini-RAG는 생성AI 호출 레이어 — 역할이 다름
- **현재 vault_rag.py가 이미 유사 기능 보유** (sentence-transformers + numpy 벡터 검색)
- **권고**: Gemini-RAG는 `GEMINI_API_KEY` 등록이 가장 빠른 해결책. AutoRAG는 vault_rag.py 하이브리드 검색 업그레이드 시 참고
- Windows 제약: Docker 권장, Python 3.10~3.12만 지원
