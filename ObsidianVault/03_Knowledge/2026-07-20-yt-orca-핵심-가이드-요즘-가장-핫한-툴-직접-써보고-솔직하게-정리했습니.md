# Orca 핵심 가이드: 요즘 가장 핫한 툴, 직접 써보고 솔직하게 정리했습니다

![thumbnail](https://i.ytimg.com/vi/QqLInsPlt4Y/maxresdefault.jpg)

## 요약

요즘 가장 핫한 도구, Orca ADE를 직접 써보고 정리했습니다.

IDE와 터미널을 하나로 합친 도구 오르카(Orca)
클로드 코드 같은 AI 코딩 에이전트를 여러 개 동시에 굴릴 수 있는
ADE(Agent Development Environment)라는 카테고리의 도구입니다.

설치부터 핵심 개념인 Git Worktree, 몇 가지 주요 기능들,
그리고 직접 써보면서 느낀 솔직한 장단점까지 담았습니다.

🐳 Orca 공식 사이트: https://www.onorca.dev
🐳 GitHub: https://github.com/st
