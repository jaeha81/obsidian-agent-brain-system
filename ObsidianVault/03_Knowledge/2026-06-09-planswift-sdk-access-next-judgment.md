# PlanSwift SDK 접근과 다음 판단

## 개요

PlanSwift는 건설·인테리어 분야의 적산(takeoff) 및 견적 소프트웨어다.  
이 노트는 PlanSwift SDK 접근 가능성과 JH 견적 자동화 시스템 연결 판단을 위한 지식 후보로 등록되었다.

## 승인 상태

- **승인일**: 2026-06-09
- **승인 출처**: Daily Plus 대시보드 사용자 직접 승인
- **큐 상태**: 등록 완료 — 우선순위 확보 시 실행 큐로 승격

## 다음 판단 기준

PlanSwift SDK 활용을 검토할 때 확인할 항목:

1. SDK/API 공개 여부 및 접근 방법 (공식 문서 URL)
2. 라이선스 조건 (상업적 사용 가능 여부)
3. 출력 형식 (JSON/XML/CSV 적산 데이터)
4. JH 견적 자동화 파이프라인 연결 가능성
5. 대안 도구 비교 (Bluebeam, PlanGrid, ProEst 등)

## 연결 가능 프로젝트

- Wishket 건설/인테리어 견적 자동화 의뢰 수주
- JH 내부 견적 생성 파이프라인 (`scripts/wishket_development_request.py` 확장)
