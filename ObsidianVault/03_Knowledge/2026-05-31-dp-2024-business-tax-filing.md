# 2024 개인사업 정산 신고

> ChatGPT Pulse 2026-05-31 Card 11 증류 (P3 · knowledge-candidate)

## 핵심 원칙

폐업신고를 해도 폐업 전 소득에 대한 종합소득세 신고 의무는 사라지지 않는다. 기타소득도 일정 요건 초과 시 신고 대상이다.

## 신고 대상 소득 유형

| 소득 유형 | 신고 대상 여부 | 비고 |
|---------|------------|------|
| 사업소득 (폐업 전) | 반드시 신고 | 폐업일까지의 수입 포함 |
| 근로소득 | 연말정산 또는 종합소득세 신고 | 타 소득 있을 경우 합산 |
| 기타소득 | 연 300만원 초과 시 신고 의무 | 원천징수 여부 무관 |
| 금융소득 | 연 2,000만원 초과 시 종합과세 | 이자+배당 합산 |
| 임대소득 | 주택임대 연 2,000만원 초과 시 | 분리과세 또는 종합과세 선택 |

## 신고 기한

```
종합소득세 신고: 매년 5월 1일 ~ 5월 31일
  ↓
성실신고 확인 대상 사업자: 6월 30일까지
  ↓
기한 후 신고: 신고 안 한 경우 가산세 부과
  (무신고 가산세: 납부세액의 20%)
```

## 세금 계산 방법

```
종합소득금액 = 사업소득 + 기타소득 + 근로소득 + ...
  ↓
과세표준 = 종합소득금액 - 소득공제 (인적공제, 연금보험료 등)
  ↓
산출세액 = 과세표준 × 세율 (누진세율 적용)
  ↓
결정세액 = 산출세액 - 세액공제 (자녀세액공제, 중소기업 특별공제 등)
  ↓
납부세액 = 결정세액 - 기납부세액 (원천징수 등)
```

## 2024년 소득세율 구간 (국세청 기준)

| 과세표준 | 세율 | 누진공제 |
|---------|------|--------|
| 1,400만원 이하 | 6% | - |
| 1,400만~5,000만원 | 15% | 126만원 |
| 5,000만~8,800만원 | 24% | 576만원 |
| 8,800만~1.5억원 | 35% | 1,544만원 |
| 1.5억~3억원 | 38% | 1,994만원 |
| 3억~5억원 | 40% | 2,594만원 |
| 5억~10억원 | 42% | 3,594만원 |
| 10억원 초과 | 45% | 6,594만원 |

## 주의 사항

- 폐업 후에도 5년간 세금 추징 가능 (시효 주의)
- 장부 미작성 시 단순경비율·기준경비율 적용
- 전자세금계산서 미발행 시 가산세 2%
- 국세청 홈택스(hometax.go.kr) 또는 세무사 상담 권장

## 관련 링크

- 국세청 홈택스: https://www.hometax.go.kr
- 종합소득세 신고 안내: 홈택스 → 신고/납부 → 종합소득세
