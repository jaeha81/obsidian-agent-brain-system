# 묶음 상품으로 ARPU 키우기

> ChatGPT Pulse 2026-05-28 Card 9 증류 (P3 · knowledge)

## 목적
화이트라벨 런칭 전략으로 ARPU를 끌어올린 Simology 사례. 단순 제품 확장이 아닌 구조적 매출 레버: 번들 패키지, 파트너 채널, 반복 매출 설계. 신규 고객 획득 비용 없이 기존 고객에게 더 많은 가치를 제공해 매출 확대.

## 핵심 내용
- **Simology 사례**: 화이트라벨 파트너에게 번들 패키지 제공 → ARPU 상승
- **번들 구성 기준**:
  - 핵심 제품 + 보완 서비스 묶음 (예: SaaS + 온보딩 지원 + API 접근)
  - 가격: 개별 합산의 70~80% (체감 할인)
  - 최소 3개 티어: Starter / Pro / Enterprise
- **ARPU 계산 방식**:
  ```
  ARPU = 총 수익 / 활성 사용자 수 (월 기준)
  목표: 번들 도입 후 3개월 내 ARPU 20% 이상 향상
  ```
- **채널 파트너 설계**:
  - 파트너에게 마진 20~30% 제공
  - 화이트라벨: 파트너 브랜드로 리셀링 허용
  - 파트너 대시보드: 고객 현황, 수익 추적
- **반복 매출 설계**: 월정액 기반, 연간 결제 시 2개월 무료 (고객 락인)
