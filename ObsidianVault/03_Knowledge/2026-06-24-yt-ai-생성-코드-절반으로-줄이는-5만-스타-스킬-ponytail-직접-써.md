# AI 생성 코드 절반으로 줄이는 5만 스타 스킬, Ponytail 직접 써봤습니다

![thumbnail](https://i.ytimg.com/vi/oaENE_9D95Q/maxresdefault.jpg)

## 요약

1. **Ponytail**은 GitHub 출시 2주 만에 5만 스타를 받은 Claude Code 스킬로, AI 에이전트가 새 코드를 작성하기 전에 기존 해결책을 먼저 탐색하게 만드는 '게으른 시니어 개발자' 원칙(YAGNI·KISS·DRY)을 적용한다.
2. 모달, 색상 선택기, 아코디언, 날짜 선택기 등 **소형 컴포넌트**에서는 생성 코드량이 절반 이하로 줄었으며 토큰 소비도 함께 감소했다.
3. 단, **대규모 앱(칸반 보드 등)** 에서는 기존 코드 재사용 여지가 적어 효과가 제한적이었다 — 작은 컴포넌트 단위 작업에 가장 효과적인 스킬이다.
