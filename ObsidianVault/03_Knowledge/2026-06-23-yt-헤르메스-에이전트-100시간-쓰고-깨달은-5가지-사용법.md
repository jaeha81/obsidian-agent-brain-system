# 헤르메스 에이전트 100시간 쓰고 깨달은 5가지 사용법

![thumbnail](https://i.ytimg.com/vi/7XA5qi8Z3zE/maxresdefault.jpg)

## 요약
헤르메스 에이전트를 100시간 이상 쓴 인디해커가 자신의 실제 사용법 5가지를 공개한다. 핵심 관점은 "헤르메스는 도구가 아니라 스스로 일하는 직원에 가깝고, 내가 도구를 부리는 구조가 아니라 나를 감싸는 더 큰 테두리(세컨드 브레인)로 쓸 때 진가가 나온다"는 것이다. Notion 워크스페이스 전체와 Claude 대화·메모리 데이터를 한 폴더로 모아 위키로 구조화해 자신을 학습시키고, 메일 자동 답장·유튜브 리뷰·X/Reddit 시장 리서치·즉석 제품 프로토타이핑·가족용 프로필까지 자동화한다. 비용은 정밀 사고엔 상위 모델(GPT 5.5), 코딩·반복 루틴엔 가성비 모델(Kimi K2)로 나눠 최적화한다.

## 핵심 개념
- **에이전트 = 직원, 세컨드 브레인**: 내가 도구를 부리는 구조가 아니라 "나를 감싸는 테두리"로 쓸 때 강력. 나에 대한 거의 모든 자료를 넣고 MCP로 서비스를 연결하면 "또 하나의 나"처럼 작동. 쓸수록 강해진다(강력한 메모리·자가 발전).
- **세컨드 브레인 구축 재료**: ① Notion 워크스페이스 전체 내보내기(또는 Notion MCP 동기화) → 헤르메스 내장 LLM 위키로 AI가 읽기 좋게 구조화. ② Claude 설정 → 개인정보 보호 → 데이터 내보내기로 전체 대화 + Claude가 나를 정의한 메모리 파일까지 확보. 하나의 폴더로 통합.
- **데스크톱 vs 텔레그램**: 텔레그램은 대화가 한 덩어리로 구별이 어려움. 데스크톱은 작업마다 세션이 시각적으로 분리. 프로필(서로 다른 헤르메스: 다른 모델·다른 기억), 아티팩트(보낸 링크·파일 한 곳 확인), 스킬탭 제공.
- **추가 세팅**: 메모리 레이어 `혼초`(대화 요약이 아니라 사고방식·말투를 학습해 사람 자체를 모델링), VPS + 리모트 게이트웨이로 로컬이 아니어도 데스크톱/웹 사용.
- **비용 최적화(모델 라우팅)**: 깊은 사고는 GPT 5.5·Claude 상위 모델, 코딩·반복 작업은 Kimi K2 같은 가성비 모델. GPT 5.5만 쓰던 달엔 API 200달러 → Kimi로 대체해 대폭 절감. `/model` 또는 "이번 작업은 키미로" 지시로 교체.
- **사용 사례 5가지**: ① 메일 자동 답장(Gmail 연동, 답장 필요 여부 분류 + 내 스타일 초안 작성) ② 유튜브 리뷰(YouTube API로 스튜디오 데이터 접근, 매일 성과·방향성 리포트, 크론 세팅) ③ X/Reddit 시장 리서치(Agent Reach로 지정 계정·서브레딧·키워드만 요약 수신; Product Hunt·Indie Hackers·Hacker News도 매일 확인) ④ 즉석 제품 빌드(시장에서 본 아이디어를 Kimi K2.7 코드 하이스피드 모델로 랜딩+동작 프로토타입; 예시 AI 회의록 제품 12.6만 토큰·약 0.2~0.3달러, Opus 대비 5~6배 저렴) ⑤ 가족용 프로필(안 쓰는 아이패드 텔레그램을 거실에 두고 음성 대화·이미지/영상 생성).
- **핵심 조언**: 시장 리서치는 처음부터 볼 대상을 좁혀라(특정 서브레딧/특정 계정). 기존 프로젝트는 Codex·Claude Code로 가되, "AI와 나누던 대화·생각"은 헤르메스로 옮겨라 — 공동 창업자에 가깝게 사업 전체를 레버리지.

## 언급된 도구/프레임워크
- `헤르메스 (Hermes) 에이전트` — 스스로 일하는 직원형 에이전트. 강력한 메모리·자가발전, 프로필·아티팩트·스킬탭, LLM 위키 내장.
- `Kimi K2 (K2.6 / K2.7 코드 하이스피드)` — 가성비 코딩·반복 루틴용 오픈소스 모델. Opus 대비 약 1/5~1/6 비용.
- `GPT 5.5` — 깊은 사고용 핵심 상위 모델.
- `혼초 (Honcho)` — 사고방식·말투를 학습해 사람을 모델링하는 메모리 레이어.
- `Agent Reach` — 에이전트가 웹(X·Reddit 등 지정 계정·서브레딧·키워드)을 읽고 요약 수집하게 하는 도구.
- `Notion / Notion MCP` — 세컨드 브레인 원재료(워크스페이스 전체 내보내기·동기화).
- `MCP (Gmail·YouTube API·핵스필드 등)` — 서비스 연동. 메일·유튜브 데이터·이미지/영상 생성.
- `VPS + 리모트 게이트웨이` — 비로컬 환경에서 데스크톱/웹 구동.
- `Product Hunt / Indie Hackers / Hacker News` — 매일 시장 흐름 확인 채널.
