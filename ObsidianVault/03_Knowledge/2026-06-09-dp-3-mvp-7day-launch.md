# 3개 MVP 7일 런치 플랜

> ChatGPT Pulse 2026-06-09 Card 8 증류 (P2 · knowledge-candidate)

## 목적

현금 흐름을 빠르게 만드는 3개 MVP 실행 패키지. 비개발자 기준, 7일 안에 첫 결제까지.

## 3개 MVP 정의

### MVP A — 이부장 세팅 서비스
- **가격**: ₩299,000 초기 + ₩99,000/월
- **내용**: 이부장 에이전트 설치 + 30일 운영 코칭
- **채널**: 카카오 채널, 블로그 포스팅
- **목표 고객**: 인테리어 소규모 업체 대표

### MVP B — 견적 CSV 표준화 키트
- **가격**: ₩99,000 (1회성)
- **내용**: Google Sheets 템플릿 + Apps Script + 설치 가이드
- **채널**: 네이버 스마트스토어 또는 직접 결제 링크
- **목표 고객**: 견적 작업 담당자

### MVP C — CAD→견적 PoC 컨설팅
- **가격**: ₩500,000/프로젝트
- **내용**: DXF 파싱 PoC 구축 + 결과 보고서
- **채널**: Wishket 프로젝트 의뢰
- **목표 고객**: 건설/인테리어 중견 업체
