# 자료가 쌓일수록 똑똑해지는 AI | 옵시디언 + 헤르메스 에이전트 + Graphify로 나만의 지식 쌓기 [LLM 위키]

![thumbnail](https://i.ytimg.com/vi/hglx5cNckAo/maxresdefault.jpg)

## 요약
지난 영상에서 옵시디언 + 헤르메스 에이전트로 만든 "LLM 위키"에, 이번엔 **Graphify(그래피파이)** 라는 무료 오픈소스 도구를 더해 대량 자료를 저비용으로 빠르게 정리하는 방법을 다룬다. 핵심 통찰은 위키와 Graphify가 경쟁 도구가 아니라 **역할 분담** 관계라는 것 — 내 관점을 담아 깊이 정리할 핵심 자료는 위키(에이전트가 하나하나 요약), 양이 많아 빠르게 훑어야 할 자료는 Graphify(지식 그래프)로 처리한다. Graphify를 설치해 같은 주제 기사 20개를 그래프로 만들고, 기존 위키는 그대로 둔 채 옵시디언 별도 폴더에 "참고 자료층"으로 쌓는 실전을 시연한다.

## 핵심 개념
- **LLM 위키 / RLM 시스템**: 내가 자료를 넣으면 AI가 읽고 스스로 요약·연결·업데이트해 나만의 지식 베이스(두 번째 뇌)를 만드는 시스템. 인터넷의 뻔한 답이 아니라 내 자료 기반 맞춤 답을 돌려줌.
- **컨텍스트 윈도우 한계**가 근본 문제: 프로젝트가 커지면 코드/문서/논문 자료도 방대해지는데 AI가 한 번에 다룰 양엔 한계가 있음. 그래서 에이전트는 전체를 올리지 않고 필요 부분만 검색해 읽음(두꺼운 책 목차 비유). 자료가 커질수록 비용↑, 연결 누락↑.
- **Graphify가 하는 일**: 흩어진 자료를 AI가 한눈에 알아보는 "지도(지식 그래프)"로 만듦. 소스코드·마크다운·PDF·스크린샷·이미지 다이어그램까지 읽어, 개념들이 서로 어떻게 연결되는지를 그래프로 엮음.
- **자료 종류별 처리 방식 분리**:
  - 소스 코드 → **Tree-sitter(트리시터)** 로 구조를 그대로 파싱. **AI를 전혀 안 씀** → 비용 0, 결과 일정, 코드 외부 유출 없음. Graphify가 코드를 빠르고 저렴하게 다루는 비결.
  - 문서·PDF·논문·이미지 다이어그램 → 클로드 같은 AI가 직접 내용을 읽고 의미·개념 연결을 해석. 다이어그램은 AI가 이미지를 보고 이해.
  - 양쪽 정보를 합쳐 자주 엮이는 것끼리 "커뮤니티" 덩어리로 정리.
- **RAG와의 차이**: 보통은 자료를 잘게 쪼개 질문과 유사한 조각을 찾는 방식(청킹+검색). Graphify는 쪼개는 대신 **"연결 자체"를 지도로 그림**.
- **위키 vs Graphify 역할 분담(핵심 메시지)**: 위키는 에이전트가 자료 하나하나를 정성껏 읽고 요약+내 관점까지 담음 → 소수 핵심 자료엔 강점, 수십·수백 개엔 시간·비용 과다. Graphify는 관점을 입히는 대신 자료 더미를 통째로 빠르게 지도화 → 대량 자료에 강점. 둘은 보완 관계.
- **실전 구성**: 위키를 Graphify로 바꾸는 게 아니라, 기존 위키는 그대로 두고 Graphify 결과를 **별도 참고층**으로 추가. Graphify에는 결과물을 옵시디언으로 바로 내보내는 기능이 기본 탑재.
- **폴더 분리 팁**: Graphify로 처리할 자료는 기존 `로우` 폴더가 아니라 `로우/graphify_articles` 같은 별도 폴더에 저장 — 같은 로우 폴더에 넣으면 이미 정리된 자료까지 재처리됨.
- **설치 요건**: Python(3.10+ / 맥은 3.12) + UV(파이썬 프로그램 설치·관리 도구). 맥은 Homebrew로 `brew install python@3.12 uv`, 윈도우는 python.org에서 파이썬 설치 후 PowerShell로 UV 설치. 이후 Graphify 설치 명령 + 헤르메스에 사용법을 알려주는 스킬 파일 설치.

## 언급된 도구/프레임워크
- `Graphify` — 흩어진 자료를 지식 그래프로 엮는 무료 오픈소스 도구. 코드베이스에서 특히 강력. 옵시디언 내보내기 기본 지원.
- `Tree-sitter(트리시터)` — 소스코드 구조를 AI 없이 파싱하는 도구. Graphify의 코드 처리 엔진(저비용·결정론적).
- `옵시디언(Obsidian)` — LLM 위키의 저장 공간. obsidian.md에서 OS별 다운로드.
- `헤르메스 에이전트(Hermes Agent)` 데스크톱 앱 — 위키 구축·Graphify 조작에 쓰는 AI 에이전트. 모델 프로바이더로 노스 포탈(300+ 모델), OpenAI OAuth(ChatGPT 구독 연동), Anthropic OAuth(Max 플랜+크레딧) 선택.
- `UV` — 파이썬 프로그램 설치·관리 도구.
- `클로드 코드 / 코덱스` — 자료가 방대해지는 실제 업무 AI 도구 예시로 언급.
