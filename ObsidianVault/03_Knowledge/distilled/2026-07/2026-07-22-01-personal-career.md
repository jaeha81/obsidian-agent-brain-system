## 핵심 인사이트

- 복합 정체성: 인테리어 분야(디자이너·엔지니어·현장관리·감리·견적상담·협력사 조율) + AI/사업 분야(자동화 기획, JH Connect/JH.AI/JH Harness 구상, AI 풀스택 빌더). 고양시 거주, GitHub jaeha81.
- 일반 디자이너와의 차이: 도면·렌더링 중심이 아니라 현장 흐름·공정·견적·하자·협력사·고객 심리·시공 난이도까지 함께 판단하는 실무 핸즈온형.
- 도메인 지식이 AI 설계에 직결된다: 공종 분류 체계, 업체마다 다른 Excel 단가 구조의 정규화 문제 → JH-EstimateAI 탄생 배경. 공정 분업 경험 → 에이전트 분업 설계 방식에 직접 영향.
- AI 관계 원칙(불변): 재하가 통제하고 AI가 실행. 승인 없는 코드 생성 금지. 아부·불필요한 칭찬 거부.
- 커뮤니케이션 스타일: 직접적·사실 기반, Yes=Yes/No=No, 이미 설명한 내용 반복 요청 금지, 불가능은 불가능하다고 말하기, 약점·리스크 반드시 언급, 실행 가능한 대안 우선.
- 개발 환경: Ryzen 9 7950X / 64GB DDR5 / RX 7700 XT / 2TB 990 Pro (~286만원), 로컬 LLM 없음 — Claude Code Max API 전용. A컴·B컴·모바일을 GitHub sync로 연결. 기기: 갤럭시 S24 울트라, 탭 울트라 8, 워치.
- 회사 내 목표: 인테리어 실무 + AI 자동화 역량 결합으로 회사 수익 창출에 직접 기여하는 핵심 인물로 성장.
