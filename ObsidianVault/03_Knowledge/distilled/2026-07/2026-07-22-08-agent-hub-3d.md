> JH Agent Hub 3D — React JSX + Three.js로 17개 에이전트를 4개 그룹(하네스/브레인/견적/판매)으로 3D 시각화한 프로토타입. Next.js + Supabase Realtime 풀 프로젝트 전환이 다음 단계.

## 핵심 인사이트

- `jh-agent-hub-3d.jsx` — React JSX + Three.js 단일 파일 프로토타입, 17개 에이전트 시각화. 상태: Three.js 프로토타입 완료.
- 4개 그룹 구성: **하네스**(제어/오케스트레이션), **브레인**(지식/기억 관리), **견적**(인테리어 견적), **판매**(영업/클라이언트).
- 에이전트 캐릭터는 봇 형태: 머리(구체) + 몸통(박스) + 팔(실린더). **Main 에이전트 = 왕관(Crown) + 안테나(Antenna)** 장착, Sub 에이전트는 기본 형태로 구분.
- 인터랙션: 그룹 단위 드래그 이동, 카메라 회전(Rotate) + 줌(Zoom).
- 연결선 3타입: **Flow**(작업 흐름)=실선, **Sync**(동기화)=점선, **Data**(데이터 전달)=파티클 애니메이션.
- Next Steps: ① 런타임 에이전트 추가/삭제 UI ② 에이전트 간 메시지 애니메이션 ③ Next.js + Supabase 풀 프로젝트 전환 — research.md → plan.md → "구현해" 워크플로우 적용, Supabase Realtime으로 실제 에이전트 상태 반영.
- 연관: JH-하네스(시각화 대상 구조의 설계 기반), JH-브레인시스템(브레인 그룹), JH-EstimateAI(견적 그룹).
