# 3개 MVP 7일 런치 플랜

Cash flow 최우선. 비개발자가 7일 내 첫 결제를 받을 수 있도록 설계된 3개 MVP 패키지.

---

## MVP A — 이부장 세팅 서비스

**가격**: ₩299,000 (초기 세팅) + ₩99,000/월 (유지관리)
**타겟**: 중소 건설/인테리어사 관리자, 오너 사장님
**가치 제안**: AI 목소리 메모 → Obsidian + Discord 자동화 세팅, 직접 설정 불필요

### Day 1-7 런치 체크리스트

- [ ] **Day 1**: 랜딩 페이지 1페이지 Notion/Framer 작성 (이미지 + CTA 버튼)
- [ ] **Day 2**: Toss Payments 테스트 결제 링크 생성 (₩299,000)
- [ ] **Day 3**: 카카오 채널 또는 오픈채팅 개설 — 문의 수신 채널
- [ ] **Day 4**: 지인 3인에게 직접 DM 발송 (베타 할인 50% 제안)
- [ ] **Day 5**: 인스타그램/블로그 1개 포스팅 — "AI가 이부장처럼 회의록 씁니다"
- [ ] **Day 6**: 첫 고객 온보딩 세션 예약 확인
- [ ] **Day 7**: 결제 완료 → 세팅 시작

### 성공 기준

- 7일 내 결제 1건 이상
- 온보딩 완료 시간 < 2시간
- 고객 재구독 전환율 목표 > 60%

## MVP B — CAD→견적 자동화 파이프라인

**가격**: ₩490,000 (파이프라인 세팅) + ₩149,000/월
**타겟**: 건설/인테리어 설계사무소, 견적 담당자
**가치 제안**: CAD 도면 업로드 → AI 자동 물량 산출 + 견적서 PDF 생성

### Day 1-7 런치 체크리스트

- [ ] **Day 1**: 견적 자동화 데모 영상 60초 제작 (화면 녹화)
- [ ] **Day 2**: 유튜브/인스타 업로드 + "현재 30% 할인 얼리버드" CTA
- [ ] **Day 3**: 건설 관련 네이버 카페 1곳 홍보 포스팅
- [ ] **Day 4**: 파이프라인 MVP 배포 (Vercel + n8n 또는 Make.com)
- [ ] **Day 5**: 테스트 고객 1인 무료 체험 제공 → 후기 수집
- [ ] **Day 6**: 후기 기반 랜딩 페이지 업데이트
- [ ] **Day 7**: 유료 전환 요청

### 성공 기준

- 7일 내 테스트 고객 3인 이상 체험
- 견적서 생성 시간 < 5분 (기존 대비 90% 단축)
- 유료 전환 1건

## MVP C — ProSuTech Builder 콘텐츠 자동화

**가격**: ₩350,000 (초기 세팅) + ₩120,000/월
**타겟**: 1인 기업, 프리랜서, 유튜버/블로거
**가치 제안**: 음성 메모 → 숏폼 스크립트 + 블로그 초안 + SNS 포스팅 자동 생성

### Day 1-7 런치 체크리스트

- [ ] **Day 1**: 서비스 소개 원페이저 작성 (Notion 공유 링크)
- [ ] **Day 2**: 직접 시연 영상 제작 — 음성 녹음 30초 → 콘텐츠 3개 자동 생성
- [ ] **Day 3**: 프리랜서 커뮤니티(크몽/숨고) 서비스 등록
- [ ] **Day 4**: Trigger.dev 자동화 파이프라인 배포
- [ ] **Day 5**: 첫 고객 시범 서비스 제공 (1일 무료 체험)
- [ ] **Day 6**: 체험 피드백 반영 → 가격/패키지 최종 확정
- [ ] **Day 7**: 정식 판매 시작

### 성공 기준

- 7일 내 크몽/숨고 주문 1건 이상
- 콘텐츠 생성 전체 사이클 < 3분
- 고객 만족도 4.5/5.0 이상

---

## 공통 런치 원칙

1. **결제 링크 먼저**: 랜딩 페이지 완성 전에 결제 링크 생성
2. **완벽주의 금지**: 기능 60% 완성도로 판매 시작, 나머지는 고객 돈 받고 완성
3. **대화 판매**: 광고보다 직접 DM이 7일 안에 빠름
4. **증거 기록**: 매일 체크리스트 완료 여부를 Obsidian Daily Note에 기록
