# 클로드 코드 파일 읽기 토큰 99% 줄이는 codebase-memory-mcp

![thumbnail](https://i.ytimg.com/vi/QOAJYkuX_-Y/maxresdefault.jpg)

## 요약

**codebase-memory-mcp**는 코드베이스 전체를 함수·클래스·호출관계 등으로 구성된 "지식 그래프"로 인덱싱해, AI가 파일을 직접 읽지 않고 그래프에 질의만으로 코드 구조를 파악하게 한다. 개발사 측정 기준 구조 질문 5개에서 파일 읽기 대비 412,000 토큰 → 3,400 토큰으로 **99.2%(약 120배) 절감**이 가능하다. 158개 언어 지원, 단일 바이너리, 자체 LLM·API 키 불필요로 Claude Code가 해석을 담당하는 구조다.
