## 콘텐츠 전략 분석

**퍼널 구조:** 바이럴 데모(브레인 시각화, Jarvis 음성) → 호기심 자극 캡션 → Discord 무료 커뮤니티 → (추정) 유료 상품. 1기 FBA 시절의 "DM me FBA" 퍼널을 AI로 재활용한 것.

**조회수가 터진 공통 요소 (상위 6개):**

1. 시각적 데모 — 뉴런 그래프가 발화하는 화면, 음성으로 깨어나는 Jarvis. 말로 설명한 영상은 전부 저조.
2. 기존 도구 공격 — "Obsidian 버려라"(22.5K, 95.2K). 논쟁 유발 = 댓글 = 알고리즘.
3. 구체적 기능 나열 — "화면 감시 + 전부 기억 + 실시간 패턴 감지"(120.6K).

**Neurolinked가 주장하는 기능 (캡션 기반):**

- 뉴런 발화형 지식 그래프 (정적 노트가 아닌 활성 네트워크)
- 화면 실시간 관찰 (screen watching)
- 영구 기억 (remembers everything)
- 실시간 패턴 감지 (notices patterns in real time)
- 음성 기반 작업 위임 ("Jarvis, wake up" → 아침에 작업 지시)
- 비즈니스 엔드투엔드 자동화 (콜드콜, 마케팅, 이커머스)

## ⚠️ 시스템 업그레이드에 참고할 때의 리스크

1. **검증 불가** — 이 계정은 기능 데모 클립만 보여주며, 실제 작동하는 제품인지 마케팅 연출인지 확인 불가. 1년 전 FBA 코칭을 팔던 동일 퍼널 구조라는 점이 신뢰도를 낮춤.
2. **기술 사상은 차용 가치 있음** — "정적 노트 → 능동적 브레인(기억+패턴감지+자동실행)" 방향성 자체는 현재 폴더(obsidian-agent-brain-system)의 업그레이드 방향과 일치. 구현은 Claude Code 기반으로 충분히 재현 가능.
3. XRP 매수 권유 이력 등 금융 조언 신뢰성 없음.
