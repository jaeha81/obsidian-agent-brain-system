# 영업 이메일 템플릿 (문을 여는 한 줄)

> E5: 첫 접촉을 열어주는 이메일 — 인테리어 사업자, AI 자동화 수요처, Wishket 잠재 고객 대상

---

## 핵심 원칙

> 1줄로 호기심을 만들고, 2줄로 증거를 대고, 1줄로 요청한다.

- 나 → 상대방 관점으로 작성 ("당신의 문제"를 먼저)
- 구체적 숫자/결과 포함
- CTA는 단 하나, 쉬운 요청 (예: "5분 통화 가능하신가요?")

---

## 템플릿 A — 인테리어 사업자 (Express Mockup)

**제목**: 공간 사진 1장으로 고객 설득하는 방법 (실제 사례)

```
안녕하세요, [이름]님.

소상공인 인테리어 사업자분들이 고객 설득에 쓰는 시간을
AI 렌더링으로 80% 줄인 사례를 공유해도 될까요?

현장 사진 → 2K 리모델링 비주얼 → 24시간 내 전달.
무료 샘플 1장을 먼저 드릴 수 있습니다.

5분 통화 가능하신 날짜를 알려주시겠어요?

재하 드림
```

---

## 템플릿 B — AI 자동화 수요처 (Wishket/프리랜서)

**제목**: [프로젝트명] 자동화로 주 10시간 절약한 방법

```
안녕하세요.

[회사/서비스명]의 [반복 업무]를 자동화해
주 10시간 이상을 절약한 사례를 Wishket에서 작업했습니다.

비슷한 업무가 있으시다면 30분 미팅으로
현재 워크플로우 분석을 무료로 드릴 수 있습니다.

관심 있으시면 편한 시간 알려주세요.

재하 드림 | AI 자동화 / 인테리어 설계
```

---

## 템플릿 C — 파트너 제안 (구독 파트너)

**제목**: [파트너사] × JH AI — 수익 분배 구조 제안

```
안녕하세요, [이름]님.

[파트너사]의 고객에게 AI 인테리어 렌더링 서비스를
화이트라벨로 제공하고 수익을 30% 배분하는 제안을 드리고 싶습니다.

현재 월 [N]건 처리 중이며, 파트너사 브랜드로 리셀링이 가능합니다.
초기 셋업 비용은 없습니다.

관심 있으시면 자료 먼저 보내드릴게요?

재하 드림
```

## 후속 이메일 (2차 접촉, 5일 후 무응답 시)

```
안녕하세요, [이름]님.

지난 이메일 한 번 더 말씀드립니다.
아직 관심 없으시면 괜찮습니다.

만약 지금은 아니더라도 나중에 [무료 샘플/자료]가 필요하시면
언제든 연락주세요.

재하 드림
```
