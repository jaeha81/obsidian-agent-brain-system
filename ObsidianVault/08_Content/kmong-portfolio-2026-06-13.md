## 서비스 1 — 업무 자동화 구축

**서비스명:**
```
GPT·Claude 연동 업무 자동화 구축 (엑셀·Google Sheet·Make·Zapier)
```

**서비스 한 줄 소개:**
```
매일 반복되는 업무를 AI가 대신 처리하도록 자동화해 드려요. 견적·보고서·데이터 집계·알림까지.
```

**서비스 상세 설명 (양식란 전체):**
```
■ 서비스 개요

엑셀, Google Sheet, Make.com, Zapier를 ChatGPT/Claude API와 연동해
매일 반복하는 업무를 자동화합니다.

저는 직접 Bucky AI 자동화 시스템을 운영 중인 실무자입니다.
이론이 아니라, 매일 돌아가는 실제 시스템을 만들어 드려요.

■ 포함 작업

• 반복 입력 업무 자동화 (데이터 수집·정리·분류)
• AI 기반 문서 자동 생성 (견적서·보고서·요약본)
• Google Sheet + ChatGPT/Claude API 연동
• 자동 알림 발송 (이메일·슬랙·카카오·Discord)
• Make.com / Zapier 워크플로우 구성
• 운영 중 오류 모니터링 + 로깅

■ 납품 결과물

• 완성된 자동화 시트 또는 스크립트
• 사용 가이드 (영상 또는 PDF)
• 1개월 무상 유지보수

■ 특화 분야

인테리어·건설·소상공인 업무 (견적 자동화, 자재 비교, 현장 일지, 발주서 작성)
일반 사무 자동화 (보고서·재고관리·고객DB 정리)

■ 작업 진행 순서

1. 무료 상담 (어떤 업무를 자동화하고 싶은지 30분 통화)
2. 견적 + 진행 일정 안내
3. 자동화 설계서 공유 (1~2일)
4. 구축 + 테스트
5. 납품 + 사용 가이드 제공
```
