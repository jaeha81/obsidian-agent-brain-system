AI Development Journey

직접 구축한 AI 개발 타임라인

2022 · Phase 1
AI 도구 도입 & 업무 적용 시작

ChatGPT·Notion AI를 현장 업무에 최초 적용. 견적서 자동 초안, 이메일 요약, 자재 리스트 생성부터 시작해 AI 활용 가능성 검증.

2023 · Phase 2
Claude API 기반 에이전트 시스템 설계 착수

Claude API와 Python으로 첫 에이전트 프로토타입 개발. Obsidian 지식베이스 연동, Discord Bot 구축, 자동화 파이프라인 설계. 에이전트 간 역할 분담 구조 정립.

2024 · Phase 3
Bucky OS — 멀티 에이전트 운영체제 완성

오케스트레이터(Bucky) + 구현(Claude Code) + 검수(Codex) 3층 구조 완성. 6,000개 이상 지식 노드, Context Pack 관리 시스템, 자동 일일 보고, 태스크 큐 운영.

2025–현재 · Phase 4
수익화 확장 — Wishket 수주 + 스나이퍼 플랫폼

Wishket 플랫폼을 통한 AI 자동화 프리랜서 수주 본격화. 스나이퍼 구매대행 MVP 배포 완료 (Vercel). AI Spend 통합, 지식 그래프 3D 시각화, Mission Control 목표 트래킹 구현 중.
