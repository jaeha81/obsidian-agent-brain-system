운영 역할 구분

태스크보드는 실행 작업 관리를 맡고, 실제 개발/실행 상태를 추적합니다.

체크리스트는 확인/승인/재개 목록으로 보고, 사용자 검토나 다음 세션 재개 대상을 정리합니다.

Task operations channel: jh-태스크보드 shared channel을 쓰고, Checklist는 확인/재개 요청을 보냅니다.

Execution handoff: 실제 실행은 Task Board가 맡고, Checklist는 다음 세션 재개와 승인 확인을 정리합니다.
