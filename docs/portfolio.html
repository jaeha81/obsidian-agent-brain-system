유즈케이스

이런 문제를 해결합니다

건설·부동산·인테리어·AI 자동화 분야의 실제 문제 해결 경험

USE CASE 01

현장 문서 자동화

견적서·시방서·하자 보고서를 AI가 초안 생성. 설계사가 검토·확정하는 Human-in-the-loop 흐름으로 품질을 유지하면서 작성 시간 80% 단축.

USE CASE 02

고객 상담 자동화

모델하우스·분양 현장에서 방문객 질의를 AI가 1차 응답. 자재 옵션, 가격 범위, 공기(工期) 안내를 24시간 자동화.

USE CASE 03

수주 파이프라인 자동화

프리랜서 플랫폼 공고 모니터링 → 적합도 분석 → 제안서 생성 → 응찰 기록까지 완전 자동화. 23/23 테스트 통과.

USE CASE 04

멀티 에이전트 운영

Claude, Codex, 도메인 특화 에이전트를 단일 Discord 채널에서 통합 관리. 복잡한 작업의 병렬 처리와 결과 통합.

USE CASE 05

지식 베이스 & RAG 구축

Obsidian 기반 지식 관리 시스템에 AI 자동 분류·태깅·요약 파이프라인 구축. 200+ 노트 자동 처리 완료.

USE CASE 06

e-Commerce 자동화

해외직구 상품 자동 검증·마진 계산·통관 리스크 분석. 8개 지표 스코어링으로 60점 이상 상품만 선별 등재.
