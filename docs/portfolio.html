이런 문제를 해결합니다

건설·부동산·인테리어 현장의 실제 문제를 AI로 해결한 경험

01

현장 문서 자동화

견적서·시방서·하자 보고서를 AI가 초안 생성. 설계사가 검토·확정하는 Human-in-the-loop 흐름으로 품질 유지.

02

고객 상담 자동화

모델하우스·분양 현장에서 방문객 질의를 AI가 1차 응답. 자재 옵션, 가격 범위, 공기(工期) 안내 자동화.

03

수주 파이프라인 자동화

프리랜서 플랫폼 공고 모니터링 → 적합도 분석 → 제안서 생성 → 응찰 기록까지 자동화된 수주 파이프라인.

04

멀티 에이전트 운영

Claude, Codex, 도메인 특화 에이전트를 단일 Discord 채널에서 통합 관리. 복잡한 작업의 병렬 처리.

05

지식 베이스 구축

Obsidian 기반 지식 관리 시스템에 AI 자동 분류·태깅·요약 파이프라인 구축. 사내 RAG 시스템의 토대.

06

데이터 수집 & 분석

AI 플랫폼 트렌드 자동 수집, 시장 공고 분석, KPI 대시보드 자동 갱신. 데이터 기반 의사결정 지원.
