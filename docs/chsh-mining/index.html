§3. 자동 스케줄 (목표 상태)

01:30 KST → Evolution 에이전트 → #jh-chsh-mining 알림
02:00 KST → 일일 파이프라인 → #jh-chsh-mining 알림
07:00 / 12:00 / 18:00 → 업로드 실행
23:00 KST → 수익 집계 → #jh-chsh-mining 리포트

스케줄은 로컬 PC의 Windows Task Scheduler가 실행한다 (setup_scheduler.ps1).
